# Editor Notes

This section is the running editorial workspace for **The Way Translation**.

Use this area to track:

1. **inspirations/**
   - Sources, people, ideas, and references worth thinking about.
   - Not necessarily immediate action items.
   - Keep short notes with source/title/date and why it may matter.

2. **proposed-rules/**
   - Proposed translation/editorial rules.
   - Each entry should include:
     - proposed timestamp (UTC)
     - proposer
     - rule statement
     - rationale (why this rule is being proposed)
     - status (proposed/accepted/rejected/revised)

3. **reflections/**
   - Ongoing editor reflections while reading and revising.
   - Use for "what feels off," improvement ideas, textual concerns, and style tensions.
   - Keep entries dated so evolution over time is visible.

## File/entry format (recommended)

Use one markdown file per month per folder, for example:

- `inspirations/2026-04.md`
- `proposed-rules/2026-04.md`
- `reflections/2026-04.md`

Append new entries at the top with this structure:

```md
## 2026-04-08 11:29 UTC — <short title>
- Context:
- Note:
- Why it matters:
- Next step (optional):
```

## Collaboration note

When changes are discussed in chat, append the relevant note to the correct folder so editorial reasoning is preserved with timestamps.
