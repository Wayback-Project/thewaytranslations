# Rendered Documents History

This folder stores timestamped snapshots of rendered outputs.

## Workflow

1. Current working output lives in `current-form-documents/`.
2. Before replacing current output with a new render, copy the previous current set into a new UTC timestamp folder here.
3. Add one line entry to `LOG.md` with timestamp, snapshot folder, and rationale.

## Snapshot naming

Use UTC:

- `YYYY-MM-DD_HHMMUTC/`

Example:

- `2026-04-08_1210UTC/`
